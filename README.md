# 🧪 achievements-lab

A small sandbox for experimenting with GitHub's workflow and unlocking profile achievements.

**English** · [Türkçe](README.tr.md)

</div>

---

This is a personal playground repository — not a library or an app. It's used to practice GitHub's collaboration flow (branches, pull requests, merges) and to unlock a few GitHub profile achievements along the way.

The contents are intentionally minimal — a couple of short lab notes (`note-1.md`, `note-2.md`), each added through its own pull request. If you're looking for shippable code, there isn't any here, and that's by design.
