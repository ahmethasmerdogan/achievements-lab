# 🧪 achievements-lab

GitHub akışını denemek ve profil başarımlarının kilidini açmak için küçük bir deneme alanı.

[English](README.md) · **Türkçe**

</div>

---

Bu bir kütüphane ya da uygulama değil; kişisel bir deneme deposu. GitHub'ın iş birliği akışını (dallar, pull request'ler, birleştirmeler) denemek ve bu sırada birkaç GitHub profil başarımının kilidini açmak için kullanılıyor.

İçerik bilinçli olarak minimal tutuldu — her biri kendi pull request'i ile eklenmiş birkaç kısa laboratuvar notu (`note-1.md`, `note-2.md`). Çalıştırılabilir kod arıyorsanız burada yok; bu da bilerek böyle.

[Claude Code](https://claude.com/claude-code) ile oluşturuldu.
